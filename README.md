# SEI Creator Fund Applications Viewer

This website displays responses from applicants to the SEI Creator Fund. Users can browse, search, and filter projects by round and approval status (Accepted, Rejected, Pending).

### Features:
- View applications from different funding rounds
- Search for projects by name
- Filter by approval status
- Expand projects to see detailed responses
- Clickable project website links
- Links to recipient wallets on SEITrace

Built with **React** and **Tailwind CSS**.

To run locally:
```sh
npm install
npm start
